## IOS
### 1. flutter SDK 설치 
- [flutter install](https://flutter.dev/docs/get-started/install)

### 2. 프로젝트 clone
```
git clone https://github.com/manguuu/howdrink
```

### 3. 아이폰 pc와 연결 후 run
```
flutter run --release
```

_cf) 애플 개발자 계정으로 xcode를 로그인해야 설치가 가능합니다._
